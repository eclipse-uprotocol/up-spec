== Overview 

uProtocol Header attributes that are common for all transports. The UAttributes message holds routing and UMessage attributes such as priority, permission, communication status and more

NOTE: Please refer to https://github.com/eclipse-uprotocol/up-core-api/blob/main/uprotocol/uattributes.proto[uattributes.proto] for the latest version of the UAttributes data model
